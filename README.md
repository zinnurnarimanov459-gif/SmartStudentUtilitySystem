⚠️ Eslatma
Foydalanuvchi noto‘g‘ri qiymat kiritsa (string o‘rniga int), dastur xatolik berishi mumkin
int.Parse() va double.Parse() ishlatilgan — xavfsizlik uchun TryParse() ishlatish tavsiya etiladi
🛠️ Yaxshilash g‘oyalari

Cheksiz sikllar o‘rniga qayta menyuga qaytish
GUI (Windows Forms yoki WPF) qo‘shish
Parol kuchliligini yanada mukammal tekshirish
